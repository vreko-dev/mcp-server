## Why This Happens

AI coding assistants are powerful but not perfect:

- They don't know your commit history
- They can't see your test results
- They optimize for "looks right" not "works right"
- Mass edits happen faster than you can review

**Git doesn't know when AI touched your code. SnapBack does.**

## The Pattern

Every save above follows the same pattern:

```
1. Developer uses AI assistant
2. AI makes changes that look reasonable
3. Changes break something subtle
4. Discovery happens minutes/hours/days later
5. Without SnapBack: Hunt through history, manually revert
6. With SnapBack: One click, instant restore
```
