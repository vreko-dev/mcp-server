# Using SnapBack with Windsurf

Windsurf is an AI-powered IDE built on VS Code. SnapBack automatically
detects Windsurf's AI features and protects your code.

## Automatic Detection

SnapBack recognizes Windsurf through:
- Extension ID detection
- AI edit pattern recognition
- Multi-file change tracking

No configuration needed—install both and you're protected.

## Detection Coverage

| Mode | What's Detected |
|------|-----------------|
| Cascade AI edits | Large generations and refactors |
| Code completions | Inline suggestions |
| Multi-file refactors | Bulk operations |

## Installation

Since Windsurf is VS Code-based, installation is identical:

1. Open Windsurf
2. Go to Extensions (`Cmd+Shift+X`)
3. Search "SnapBack"
4. Click Install

## Known Considerations

- Windsurf's proprietary Cascade features may have slightly different detection patterns
- Some operations aren't distinguishable from manual edits
- We're actively improving Windsurf detection based on Pioneer feedback
