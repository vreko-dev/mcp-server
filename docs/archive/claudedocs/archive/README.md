# Archive

## Purpose

This directory contains historical documents that are no longer actively maintained but are preserved for reference. These documents provide context for past decisions and work.

## Organization

Documents are organized by date in subdirectories:

-   [2024-09-30/](./2024-09-30/) - Initial analysis and migration planning

## Standards

Archived documents should:

-   ✅ Include context about why they were archived
-   ✅ Link to superseding documents when applicable
-   ✅ Be clearly dated
-   ✅ Be preserved in their original form
