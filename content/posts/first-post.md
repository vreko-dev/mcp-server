# Welcome to Our Blog

This is our first blog post! We're excited to share updates about SnapBack with our community.

## What is SnapBack?

SnapBack is an AI-aware code protection system designed to automatically create intelligent snapshots before AI coding assistants modify your codebase.

## Key Features

- **Intelligent File Protection**: Watch, warn, or block file modifications
- **Smart Snapshots**: Content-based deduplication and full-text search
- **Privacy-First Design**: Only metadata is transmitted, not file contents
- **Multi-Platform Support**: VS Code, CLI, Web SDK, and MCP integration
