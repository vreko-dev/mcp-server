## Quick Links

### For Free Tier Users

If you're using SnapBack on the Free plan, you want the **Local MCP Integration**:

👉 [Go to Local MCP Docs →](/docs/capabilities/mcp-local)

**Features:**
- 100% offline operation
- No cloud connectivity required
- Complete privacy (data never leaves your machine)
- Available MCP tools: `create_snapshot`, `analyze_risk`, `check_deps`

### For Solo/Team/Enterprise Users

If you're on a paid plan, you can use **both** local and backend MCP:

👉 [Go to Backend MCP Docs →](/docs/capabilities/mcp-backend)

**Additional Features:**
- Cloud backup and restore
- Advanced Guardian AI risk scoring
- Team snapshot sharing (Team+ only)
- API authentication
- Higher rate limits
