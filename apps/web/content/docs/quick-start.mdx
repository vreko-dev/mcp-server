# Quick Start Guide

Get SnapBack up and running in your project in under 5 minutes.

## Installation

<div className="steps my-6">

### Step 1: Install the Extension

Open VS Code and install the SnapBack extension:

1. Press `⌘+Shift+X` (Mac) or `Ctrl+Shift+X` (Windows/Linux)
2. Search for "SnapBack"
3. Click **Install**

<div className="callout alert-info my-4">
  <strong>💡 Pro Tip:</strong> Make sure you're using VS Code version 1.70 or later for optimal compatibility.
</div>

### Step 2: Open Your Project

Open any project folder in VS Code. SnapBack will automatically activate and start protecting your code.

### Step 3: Make Your First Snapshot

Edit and save any file. SnapBack creates automatic snapshots on every save!

```typescript
// Edit this file and save - SnapBack has your back!
const greeting = "Hello SnapBack";
console.log(greeting);
```

<div className="my-4">
  <button className="copy-button bg-emerald-900/30 hover:bg-emerald-800/50 text-emerald-400 px-4 py-2 rounded-lg border border-emerald-700/50 transition-all">
    Copy Code
  </button>
</div>

</div>

## Verify Installation

Check that SnapBack is running:

1. Look for the 🧢 icon in your VS Code status bar
2. Open the SnapBack panel: `View → SnapBack`
3. You should see your first snapshots appear as you save files

<div className="callout alert-info my-6">
  <strong>💡 Pro Tip:</strong> SnapBack works automatically - no configuration needed! Your first snapshot is created as soon as you save a file.
</div>
