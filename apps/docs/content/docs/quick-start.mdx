# Quick Start Guide

**Goal:** You'll create and restore your first snapshot in under 5 minutes. No configuration needed.

## Step 1: Install the Extension (2 min)

Open VS Code and install SnapBack:

1. Press `⌘+Shift+X` (Mac) or `Ctrl+Shift+X` (Windows/Linux)
2. Search for "SnapBack"
3. Click **Install**
4. Wait for the install to complete (you'll see a notification)

<div className="callout alert-info my-4">
  <strong>💡 Pro Tip:</strong> VS Code 1.70+ recommended. If you're on an older version, update VS Code first via Help → Check for Updates.
</div>

## Step 2: See Your First Snapshot (1 min)

Edit any file and save it:

1. Open a file in your project (any file: `.ts`, `.js`, `.md`, doesn't matter)
2. Make a tiny change (add a comment, change one word)
3. Save the file (`Cmd+S` or `Ctrl+S`)

**What happens:**
- SnapBack automatically captures a snapshot of your file
- A notification appears at the bottom of VS Code
- You'll see a blue icon (🧢 SnapBack) in your status bar

<div className="callout alert-success my-4">
  <strong>✅ Success!</strong> You just created your first snapshot. SnapBack did this automatically on save.
</div>

## You're Now Protected

**Git doesn't know when AI touched your code. SnapBack does.**

From this moment, every AI-assisted change is automatically captured.
If something breaks, you're one click away from recovery.

## Verify Installation

Check that SnapBack is running:

1. Look for the 🧢 icon in your VS Code status bar
2. Open the SnapBack panel: `View → SnapBack`
3. You should see your first snapshots appear as you save files

<div className="callout alert-info my-6">
  <strong>💡 Pro Tip:</strong> SnapBack works automatically - no configuration needed! Your first snapshot is created as soon as you save a file.
</div>
