## Service Overview

The API is unified, covering both core application logic and authentication services.

- **Authentication**: Endpoints for signing in, signing up, and managing sessions (via Better Auth).
- **Telemetry**: Endpoints for analytics and identification (e.g., `/api/identify`).
- **Webhooks**: Handlers for Stripe and PostHog integration.

> [!NOTE]
> The interactive documentation merges all service schemas into a single view for easier discovery.
