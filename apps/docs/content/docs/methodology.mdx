# Detection Methodology

This page explains how SnapBack measures detection accuracy and what our claims mean.

## How Detection Works

SnapBack uses multiple signals to detect AI-assisted changes:

1. **Extension presence** — Detects active AI extensions (Cursor, Copilot, etc.)
2. **Edit patterns** — Recognizes AI-typical edit signatures
3. **Timing analysis** — Identifies rapid multi-file changes
4. **Content heuristics** — Patterns in generated code

## What "Accuracy" Means

When we say SnapBack "catches common patterns," we mean:

| Metric | Definition |
|--------|------------|
| **True positive** | AI made change, SnapBack detected it |
| **False positive** | Human made change, SnapBack flagged as AI |
| **True negative** | Human made change, SnapBack didn't flag |
| **False negative** | AI made change, SnapBack missed it |

We optimize for **low false negatives** (catching AI changes) while accepting some false positives (extra snapshots are cheap).

## Testing Methodology

### Synthetic Testing

We test detection against controlled scenarios:
- Known AI tool outputs
- Simulated editing patterns
- Edge cases (mixed human/AI edits)

### Opt-In Telemetry

Pioneers who enable detection improvement share:
- Which AI tool was active (e.g., "Cursor")
- File type affected (e.g., ".ts")
- Whether a restore occurred

This helps us validate detection in real-world conditions.

## What We Don't Claim

We deliberately avoid:
- Specific percentage accuracy claims
- Guarantees about catching every AI change
- Claims about specific AI tools being "better" or "worse"

Detection improves over time as:
- More patterns are identified
- Your codebase patterns are learned
- Community contributions add new signatures

## Improving Detection

### Report False Negatives

If AI made a change that SnapBack missed:

1. Note which AI tool was active
2. Describe what kind of change was made
3. [Report via GitHub](https://github.com/marcelle-labs/snapback/issues)

### Report False Positives

If SnapBack flagged a manual edit as AI:

1. Click "Not AI" in the snapshot viewer
2. This feedback improves local detection
