## Phase 8: Orphan Detection

### 8.1 Dead Code Identification
```bash
# Find exported functions never imported elsewhere
# Use ts-prune or knip for this

npx knip --include-entry-exports 2>/dev/null || echo "Install knip: npm i -g knip"

# Manual check for common patterns
grep -rn "export function\|export const\|export class" packages --include="*.ts" | \
  while read line; do
    fn=$(echo "$line" | sed 's/.*export \(function\|const\|class\) \([a-zA-Z_]*\).*/\2/')
    count=$(grep -r "$fn" packages apps --include="*.ts" --include="*.tsx" | grep -v "export" | wc -l)
    if [ "$count" -lt 2 ]; then
      echo "ORPHAN: $fn (used $count times)"
    fi
  done
```

### 8.2 Orphaned Components
```bash
# Find React components never imported
for file in $(find apps/web -name "*.tsx" | grep -v "test\|stories"); do
  component=$(basename "$file" .tsx)
  imports=$(grep -r "import.*$component\|from.*$component" apps/web --include="*.tsx" | wc -l)
  if [ "$imports" -eq 0 ]; then
    echo "ORPHAN COMPONENT: $file"
  fi
done
```

## Severity Definitions

| Level | Definition | UAT Impact |
|-------|------------|------------|
| **BLOCKER** | Flow completely broken, no workaround | Cannot proceed |

| **HIGH** | Feature works but with visible gaps | User notices issue |
| **MEDIUM** | Edge case failures, cosmetic issues | Acceptable for beta |
