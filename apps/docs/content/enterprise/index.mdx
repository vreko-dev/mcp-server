## Enterprise Capabilities

### Security & Privacy

- **Data Sovereignty**: Custom data residency and retention policies
- **Encryption**: End-to-end encryption for cloud backups
- **Audit Logs**: Immutable audit trails for compliance
- **DPA**: Data Processing Agreement available on request
- **Compliance**: GDPR-compliant data handling; SOC 2 Type II certification in progress

[Learn more about Security & Privacy →](/enterprise/security-privacy)

### SSO & Identity Management

- **SAML/SSO**: Integrate with Okta, Azure AD, Google Workspace, OneLogin, Auth0
- **SCIM Provisioning**: Optional automatic user provisioning and deprovisioning
- **Attribute Mapping**: Flexible user attribute mapping
- **Group Sync**: Automatic team membership based on IdP groups

[Learn more about SSO/SAML →](/enterprise/sso-saml)
