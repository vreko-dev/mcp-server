## Features

- **3-Level Protection** - Granular file-level control
- **Session Rollback** - Multi-file atomic restore
- **Guardian Detection** - 94% Day 1 accuracy that improves as it learns your codebase
- **MCP Integration** - Works with Claude Code & Cursor
- **Performance** - &lt;200ms snapshots, &lt;10ms checks
- **Privacy First** - Metadata-only transmission, never your code
